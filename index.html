Master Thesis Abstract

This study aims to develop a decision support system (DSS) to solve the multi-objective scheduling problem in universities that use credit hours. This system uses the new integrated zero-one goal programming (ZOGP) - analytic hierarchy process (AHP) approach. The objective of this system is to build a semester study schedule that optimizes achievement of personal preferences considering their importance, capable of involving additional criteria. Moreover, the system simultaneously avoids breaching all registration restrictions defined, thus, the outcome is a schedule that is rapid, ready, and optimum with respect to the predefined goals and priorities. The DSS is represented by a computer application that serves as a linear programming models generator. This application generates different multi-objective optimization models for different cases, solves them, translates the solution, and presents it in a readable format.
